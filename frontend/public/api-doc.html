🎯 阈值号码 API

Base URL: https://ct256.cn/number-warehouse/api/threshold  |  需 Bearer Token 认证

1. 计算阈值号码

POST /compute

触发阈值计算，从最新抽签日+1天的快照数据中，按 sim_results 值排序取 Top25 / Bottom24。

请求

参数类型必填说明
datestring否目标日期 (YYYY-MM-DD)。不传则自动取最新抽签日+1
# 自动取最新抽签+1 curl -X POST https://ct256.cn/number-warehouse/api/threshold/compute \ -H "Authorization: Bearer <token>" # 指定日期 curl -X POST "https://ct256.cn/number-warehouse/api/threshold/compute?date=2026-07-14" \ -H "Authorization: Bearer <token>"

响应

{ "ok": true, "date": "2026-07-14", "results": [ {"collection_id": -23, "threshold": 25, "summary_name": "B级汇总1", "count": 25}, {"collection_id": 14, "threshold": 24, "summary_name": "", "count": 24} ] }

字段说明

字段说明
collection_id见下方「collection_id 对照表」
threshold25=Top25 推荐号；24=Bottom24 避开号
summary_name汇总名（仅汇总级有值）
count数字个数

collection_id 对照表

collection_id层级名称说明
-23汇总级B级汇总1第1个汇总独立 Top25/Bottom24
-24汇总级B级汇总2第2个汇总独立
-25汇总级B级汇总3第3个汇总独立
-26汇总级B级汇总4第4个汇总独立
-28汇总级B级汇总5第5个汇总独立
-29汇总级B级汇总6第6个汇总独立
14集合级集合14前4汇总聚合 Top25/Bottom24
16集合级集合16前6汇总聚合 Top25/Bottom24
⚠️ 注意：请求会先检查 col19 快照是否存在，缺失时自动触发快照采集（含集合14/16），再执行阈值计算。整个过程可能有几秒延迟。

2. 查询阈值结果

GET /results

读取已存储的阈值号码，含 col19 汇总明细。

请求

参数类型必填说明
datestring否目标日期。不传则自动取最新数据
curl -s "https://ct256.cn/number-warehouse/api/threshold/results?date=2026-07-14" \ -H "Authorization: Bearer <token>"

响应

{ "date": "2026-07-14", "items": [ { "collection_id": -23, "summary_name": "B级汇总1", "label": "B级汇总1", "key": "s23", "thresholds": { "25": { "numbers": [3,12,25,...], "numbers_str": "3.12.25.38.7.19.31.44.2.15.27.40.9.21.34.47.5.18.30.42.11.23.36.49" }, "24": { "numbers": [1,8,14,...], "numbers_str": "1.8.14.26.39.6.17.29.41.10.22.35.48.4.16.28.37.13.24.32.43.45" } } } ], "col19_summaries": [ {"name": "B级汇总1", "total_value": 156000, "draw_number": 12345} ] }

字段说明

字段说明
items阈值条目列表，含汇总级和集合级
thresholds.25Top25 推荐数字（按 value 降序），numbers_str 为点号分隔字符串
thresholds.24Bottom24 避开数字
col19_summariescol19 当天所有汇总的总值+抽签数

3. 认证说明

所有 API 请求需携带 Bearer Token：

Authorization: Bearer <your_token>

Token 通过 POST /auth/login 获取（需密码），有效期 7 天。
